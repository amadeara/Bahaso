# Bahaso
(code ada di master)
#1. HomePage
    HomePage menampilkan artikel-artikel kesehatan yang tersedia. Untuk, melihat artikel lebih lanjut maka klik lihat artikel
#2. Artikel
    Menampilkan artikel yang ingin dilihat
#3. Button Tambah Artikel
    Jika kita ingin menambahkan artikel baru, maka kita bisa mengisi form untuk diisi seperti judul artikel, penulis, dan isi artikel yang diinginkan.
    Setelah selesai. Klik submit artikel. Halaman akan kembali ke homepage.
#4. Button update
    Untuk melihat list artikel yang sudah dibuat. Terdapat button edit dan hapus artikel.
    - edit : mengedit artikel yang diinginkan. ketika klik button submit changes, maka halaman kembali ke homepage
    - hapus : menghapus artikel yang diinginkan
